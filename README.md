# Premium Animated Portfolio (Next.js 14)

A modern, minimal portfolio template built with:

- Next.js 14 (App Router)
- Tailwind CSS
- Framer Motion
- TypeScript

## Included Features

- Animated hero section
- Typewriter job title effect
- Smooth scroll navigation
- Animated project cards
- Glassmorphism UI
- Responsive layout
- SEO metadata setup
- Clean, reusable component structure
- Ready for Vercel deployment

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
npm start
```

## Deploy to Vercel

1. Push this project to GitHub.
2. Import the repo in Vercel.
3. Keep default framework detection (`Next.js`).
4. Click deploy.

## Customize

- Update portfolio copy in `app/page.tsx` and `components/Hero.tsx`.
- Replace project items in `lib/projects.ts`.
- Change SEO defaults in `app/layout.tsx`.
